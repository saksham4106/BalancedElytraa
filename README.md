[Balanced Elytra](https://www.curseforge.com/minecraft/mc-mods/balanced-elytra) is a minecraft mod for 1.16.x versions made using MinecraftForge. 

## Features
- Elytra takes a greater durability damage during rain
- Player is more likely to be hit by lightning when flying
